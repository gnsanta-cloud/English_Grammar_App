# GitHub Pages 배포 (Safari 홈 화면 추가)

## 1. GitHub 저장소

저장소 예: [gnsanta-cloud/English_Grammar_App](https://github.com/gnsanta-cloud/English_Grammar_App)

```powershell
git init
git add .
git commit -m "Initial commit: Julia Grammar app"
git branch -M main
git remote add origin https://github.com/gnsanta-cloud/English_Grammar_App.git
git push -u origin main
```

## 2. GitHub Pages 켜기

1. **https://github.com/gnsanta-cloud/English_Grammar_App/settings/pages**
2. **Source**: **GitHub Actions**
